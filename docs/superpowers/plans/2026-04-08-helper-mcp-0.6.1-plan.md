# helper-mcp 0.6.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a large 0.6.1 release that expands Luau intelligence, brain relationships, and workspace release automation while preserving the stabilized 0.6.0 surface.

**Architecture:** Keep the current MCP registry and compatibility aliases stable, then add three new command families on top of the existing Luau/brain/workspace helpers. Each family should reuse the current snapshot/history patterns so new commands stay deterministic, auditable, and easy to regression-test.

**Tech Stack:** Node.js ESM, `node:test`, the existing MCP SDK, filesystem-backed JSON/JSONL artifacts, and the current Luau/brain/workspace modules.
