# helper-mcp 0.6.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship Luau-to-brain memory bridging so Luau findings automatically become durable brain notes with stable identities.

**Architecture:** Keep the current MCP registry stable, add a normalized Luau finding pipeline, and wire every finding-producing Luau command into a deterministic brain note bridge. Brain then exposes dedicated finding history, graph, and cleanup commands so the derived memory stays inspectable and recoverable.

**Tech Stack:** Node.js ESM, `node:test`, the existing MCP SDK, filesystem-backed JSON/JSONL artifacts, and the current Luau/brain/core modules.
