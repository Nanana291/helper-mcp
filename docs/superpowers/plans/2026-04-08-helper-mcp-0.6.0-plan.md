# helper-mcp 0.6.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a stabilization release that reduces false positives in Luau analysis and hardens recovery, persistence, and brain hygiene.

**Architecture:** Keep the existing MCP surface and compatibility aliases, but tighten the analysis path with safer pattern loading, confidence-aware findings, and stronger regression tests. Strengthen recovery and memory flows by making history, merges, changelogs, metrics, and snapshots more auditable and more deterministic.

**Tech Stack:** Node.js ESM, `node:test`, the existing MCP SDK, filesystem-backed JSON/JSONL artifacts, and the current Luau/brain/workspace modules.

```js
});

test('patterns.json only overrides the categories it defines', () => {
  const root = mkdtempSync(path.join(tmpdir(), 'helper-mcp-patterns-'));
  try {
    mkdirSync(path.join(root, '.helper-mcp'), { recursive: true });
    writeFileSync(path.join(root, 'script.luau'), 'legacyWait(0.2)\n', 'utf8');
    writeFileSync(path.join(root, '.helper-mcp', 'patterns.json'), JSON.stringify({
      risks: [{ label: 'legacy-wait', pattern: '\\blegacyWait\\s*\\(' }],
    }), 'utf8');

    const scan = scanLuauWorkspace(root);
    assert.equal(scan.files[0].categories.risks[0].label, 'legacy-wait');
  } finally {
    rmSync(root, { recursive: true, force: true });
  }
});
```

```js
  brainHistory,
  mergeBrainNotes,
  promoteBrainNote,
  updateBrainNote,
} from '../src/brain.mjs';

test('brain history records status changes and merge suggestions stay explicit', () => {
  const root = mkdtempSync(path.join(tmpdir(), 'helper-mcp-brain-'));
  try {
    appendBrainNote(root, { title: 'Wrap remotes in pcall', summary: 'Use pcall around remote calls.', scope: 'luau' });
    appendBrainNote(root, { title: 'Wrap FireServer in pcall', summary: 'Use pcall around remote calls.', scope: 'luau' });
    const notes = mergeBrainNotes(root, { noteId: 'missing', apply: false });
    assert.equal(notes.ok, false);

    const history = brainHistory(root);
    assert.ok(history.total >= 2);
  } finally {
    rmSync(root, { recursive: true, force: true });
  }
});
```
